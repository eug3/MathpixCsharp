# MathpixCsharp: C#实现的Mathpix Windows开源客户端

原版见 https://github.com/itewqq/MathpixCsharp 

![使用演示](/images/demo1.gif)

## 功能支持：

- [x] 截图识别公式转换为Latex代码
- [x] 截图识别公式转换为Office Word公式 （Word 的墨迹版本做公式也很棒，就是不能能支持 Latex 转换）
- [x] 手写公式识别支持
- [x] 多显示器支持
- [x] 最小化到系统托盘
- [x] 快捷键截图
- [x] 官方API支持，每个月可免费用500次，需要**国外信用卡**  1刀

## 客户端安装:

### 只有从源代码编译：

#### 环境准备：

- Windows 10
- .NET Framework >= 4.7.1
- Visual studio 2019
